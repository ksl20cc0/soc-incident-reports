# SOC-Incident-Reports
This repository contains simulated SOC incident response reports based on hands-on labs and threat scenarios.

## Purpose
- Practive SOC analyst reporting
- Demonstrate incident analysis and response process
- Improve written communication for SOC / IR roles

## Scope
- Phishing
- Malware detection
- Endpoint alerts
- SIEM-based triage

## Disclaimer
All reports are based on simulations.
No real organization, IP address, or personal data is used.
